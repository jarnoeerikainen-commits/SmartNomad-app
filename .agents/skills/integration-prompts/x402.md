# x402 Agentic Payments (x402)

## Secrets
- `X402_API_KEY`
- `X402_FACILITATOR_URL` (e.g. `https://x402.org/facilitator`)

## Flow
1. AI agent attempts a paid HTTP call → receives `402 Payment Required` + payment header.
2. Agent submits payment via facilitator → gets receipt → retries with `X-Payment` header.
3. Server verifies receipt with facilitator → 200.

## Activation prompt
> Wire x402 inside `agentic-payments-router`. Add `x402_handle402(response)` helper that talks to `X402_FACILITATOR_URL`. Guardrail every payment via DB rpc `evaluate_agentic_guardrail`.
